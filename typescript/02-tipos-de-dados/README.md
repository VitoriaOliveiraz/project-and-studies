# Módulo 02 — Tipos de Dados

## O que foi estudado

Aprofundamento nos tipos disponíveis no TypeScript. O módulo cobre tipos primitivos, como tipar objetos e funções, e quando usar o tipo any com responsabilidade.

## Exercícios

| Exercício                 | Descrição                                                                 |
| ------------------------- | ------------------------------------------------------------------------- |
| `01-implement-typescript` | Migração de um arquivo JS para TS, aplicando tipagem nos dados existentes |

## Conceitos praticados

- Tipos primitivos: string, number, boolean
- Tipagem de objetos
- Anotações de tipo em funções
- Diferença entre any e tipos especificos
