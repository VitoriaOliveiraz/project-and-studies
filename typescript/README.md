# TypeScript — Trilha de Estudos

## Sobre

Aqui ficam registrados os exercícios da minha trilha de aprendizado em TypeScript. Cada módulo cobre um conjunto de conceitos e tem seus próprios exercícios práticos. A ideia é ir do básico de tipagem até os recursos mais avançados da linguagem.

## Módulos

| Módulo                                      | Conteúdo                                                                       |
| ------------------------------------------- | ------------------------------------------------------------------------------ |
| [`01-introducao`](./01-introducao/)         | Primeiro contato com TypeScript, tipagem básica de variáveis, funções e arrays |
| [`02-tipos-de-dados`](./02-tipos-de-dados/) | Tipos primitivos, objetos tipados e anotações de tipo em funções               |
