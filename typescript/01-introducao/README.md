# Módulo 01 — Introdução ao TypeScript

## O que foi estudado

Primeiro contato real com TypeScript. Os exercícios cobrem tipagem de variáveis, parâmetros de função, retorno de funções, arrays tipados e o início de classes com constructor.

## Exercícios

| Exercício                | Descrição                                                          |
| ------------------------ | ------------------------------------------------------------------ |
| `01-hello-typescript`    | Primeiro arquivo em TS, tipando variáveis básicas                  |
| `02-search-book`         | Busca um livro dentro de um array e sua posição em objetos tipados |
| `03-validate-array`      | Valida se um array atende as condições de ter dados impares        |
| `04-filter-cities`       | Filtra cidades de uma lista com base em um critério                |
| `05-filter-even-numbers` | Filtra números pares de um array tipado                            |
| `06-constructor-class`   | Cria uma classe com constructor e propriedades tipadas             |

## Conceitos praticados

- Tipagem de variáveis e parâmetros
- Tipagem de retorno de funções
- Arrays tipados
- Introdução a classes com constructor
